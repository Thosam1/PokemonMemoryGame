# PokemonMemoryGame

a little memory game made using javascript (and a little bit of html and css)

methods learned : 
  • push()
  • querySelector()
  • SetAttribute()
  • getAttribute()
  • appendChild()
  • Math.random()
  • sort()
  • For loops
  • createElement()

## Usage :
run the html file and play the game

if you want to play again, just reload the page
